About Thrust RL

Thrust is a high-performance reinforcement learning library written in Rust. These demos showcase trained agents running entirely in your browser via WebAssembly. All neural network inference is done in pure Rust with no external dependencies.

Training uses PyTorch (via tch-rs) on GPU for maximum performance, then weights are exported and compiled to WASM for fast, portable inference.
